Metadata Summary

Originator Metropolitan Council - Metro Transit
Abstract The Metro Transit schedule data includes bus and LRT schedules for the Twin Cities metropolitan area. This includes data for all public transit companies in the Minneapolis / St. Paul metropolitan area except the Minnesota Valley Transit Authority. The data uses the Google Transit Feed Specification. This specification can be found at http://code.google.com/transit/spec/transit_feed_specification.html

This data feed no longer includes data for the Minnesota Valley Transit Authority (MVTA). MVTA data can downloaded here. http://www.datafinder.org/metadata/mvtaschedule.htm
